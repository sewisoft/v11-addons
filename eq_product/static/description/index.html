Equitania Produkt Optimierungen

Dieses Modul enthält Erweiterungen für den Bereich Produkte

Fügt hinzu:

  • Historie für Produktpreise (Auf der Produktmaske ist ein Link neben dem Verkaufspreis enthalten), inkl. Rechte für die Preishistorie
  • Artikelnummer in der Produktansicht
  • Nummerngenerator für Produkte
  • Anzeige der Produktbeschreibung
  • Überprüfung der Artikelnummer bei der Neuanlage eines Produkts auf Einzigartigkeit

Änderungen:

  • Verbessert die Darstellung der Produktetiketten (inkl. Barcodewechsel auf Code128)

Weitere Informationen finden Sie in unserem Wikieintrag: Modul Produkt Optimierungen
