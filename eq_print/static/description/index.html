EQ Print

Durch die Verwendung von diesem Modul haben Sie die Möglichkeit voll automatisch über Ihr Open Source ERP System Odoo zu drucken. PDF's müssen somit nicht mehr zuerst herunterladen werden, sondern werden direkt von Odoo automatisch an Ihren Drucker geschickt und somit sofort gedruckt.

Nähere Informationen zu diesem Modul finden Sie in unserem MyOdoo Wiki.
