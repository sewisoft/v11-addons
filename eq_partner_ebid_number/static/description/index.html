EBID

Dieses Modul erweitert die Partner um die Funktionen der API von Unternehmensverzeichnis.org

  • Suche eines Datensatzes im Unternehmensverzeichnis über die Adressdaten
  • Suche eines Datensatzes im Unternehmensverzeichnis über die EBID-Nummer

Weitere Informationen finden Sie in unserem Wikieintrag: Modul Unternehmensverzeichnis


Setting up the connection to Unternehmensverzeichnis.org

You need to specify the address to the webservices (the company and the matching service) and your login credentials.

Configuration of the Cronjob

Use the cronjob to either search for EBID-Numbers for addresses (Parameter (False,)) or to load company data from Unternehmensverzeichnis.org for addresses with existing EBID-Numbers (Argument (True,)).
The records that are processed by the cronjob can be limited by an additional parameter.
Example: To only process 200 records per run, use the following arguments: (False, 200)

EBID options for companies

In the form view of a company, you can manually search for the EBID-Number. The search parameters are the company name, street, house number, zip and the city. If exactly one EBID-Number was found, it will be set automatically. Otherwise a message will be shown informing you about the result.

If an EBID-Number exists, you can open the company's page on Unternehmensverzeichnis.org.

EBID Protocol

The EBID protocol contains information about the requests to Unternehmensverzeichnis.org.

If multiple possible EBID-Numbers were found for a company, the protocol's detail view allows you to choose the correct EBID-Number. Further, it offers the following option:

  • Opening Unternehmensverzeichnis.org's page for the company
  • Looking up company data from Unternehmensverzeichnis.org for the EBID-Number
  • Setting the EBID-Number for a company
  • Updating the company by using Unternehmensverzeichnis.org's data
