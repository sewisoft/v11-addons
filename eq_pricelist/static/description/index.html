Equitania Preislisten

Dieses Modul enthält Erweiterungen für den Bereich Preislisten

Aktivieren der Preislisten:

Bevor Sie Preislisten bearbeiten oder sehen können, müssen Sie diese für den Benutzer erst sichtbar schalten.
Gehen Sie hierfür in die Einstellungen > Benutzer > Benutzer
Dort müssen Sie für einen gewählten Benutzer mindestens die Einstellung Verkaufspreisliste aktivieren, es bietet sich auch gleich an die Preislistenpositionen verwalten zu aktivieren.

Nun können Sie die Preislisten unter Verkauf > Konfiguration > Preislisten finden.


Fügt hinzu:

  • Preislistenversionspositionen: Eine Maske zur Suche und einfacheren Arbeiten mit Preislistenpositionen (Verkauf > Konfiguration > Preislisten > Preislistenversionspositionen)

Weitere Informationen finden Sie in unserem Wikieintrag: Modul Equitania Preisliste
