EQ Base Report

Dieses Modul enthält Erweiterungen von Basisfunktionen und Styles für die Equitania Reports

Fügt hinzu:

  • Eine eigene Reportstyle-Datei wird nun für die Reports eingefügt
    • Standard Schriftart auf OpenSans umgestellt
    • Standard Schriftgröße angepasst
    • Abstände und Tabellenfarben angepasst
  • Separate Vorlage"Equitania" für EQ-Berichte Sie können Vorlagen einrichten von : Einstellungen > Allgemeine Einstellungen ->Geschäftsdokumente >Dokumentenvorlage
  • Verwendung eines separaten Report-Logos (statt des Standard-Firmen-Logos) für die Reports. Das Logo können Sie nun hier hochladen: Einstellungen > allgemeine Einstellungen > Geschäftsdokumente > Company Report Logo
  • Wenn das Reportlogo nicht gesetzt ist, wird ein Hinweis auf dem Report angedruckt wo und wie es anzulegen ist.
  • Folgeseiten (Seite 2+) enthalten auf den Angeboten und Rechnungen nun zusätzliche Informationen wie Auftragsnr, Kundennr, Datum oder ähnliche nötige Informationen.

Weitere Informationen finden Sie in unserem Wikieintrag: Modul Equitania Base Report
